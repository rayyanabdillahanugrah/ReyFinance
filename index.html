Konfigurasi OpenAI API

Masukkan API Key Anda untuk mengaktifkan fitur AI Financial Advisor. Key hanya disimpan di browser Anda (localStorage), bukan di server kami.

Key Anda langsung dikirim ke OpenAI.

Cara Mendapatkan API Key

  1. Kunjungi platform.openai.com
  2. Login atau buat akun OpenAI
  3. Masuk ke menu API Keys
  4. Klik Create new secret key
  5. Salin dan tempelkan key di sini
Penggunaan API dikenakan biaya oleh OpenAI. Pikirkan sebelum benar-benar menggunakan fitur ini!
